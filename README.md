Smart Menu Suggestion AI
Overview

A rule-based AI system that simulates a restaurant ordering experience. The program guides the user through menu selection, including dish category, type, portion size, drinks, and curries, calculates the total bill, and handles partial payments interactively.

Features

Interactive menu navigation based on user input.

Supports Veg, Non-Veg, and Vegan dishes.

Dynamic portion sizes with price variations.

Selection of drinks and curries with price addition.

Input validation for category, type, dish, and portion size.

Total bill calculation and payment handling with balance display.

Databases

Menu_Database: Main dishes and portion-based pricing.

Curry_MenuDatabase: Curries with Veg and Non-Veg options.

Menu_DrinkDatabase: Drinks with categories like Hot, Cold, and Fruit Juice.

How to Use

Run the Python script.

Enter your name to start the session.

Select a main dish category (e.g., Kothu, Biriyani).

Choose a food type (Veg, Non-Veg, Vegan) if applicable.

Select a dish from the suggested options.

Choose your portion size.

Optionally add drinks and curries.

Pay for your order. The program will display your final bill and any balance.
